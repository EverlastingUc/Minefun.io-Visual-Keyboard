## Overview

Minefun.io Visual Keyboard displays a virtual keyboard on your screen that highlights keys as you press them. Useful for streaming, recording, or just seeing what keys you're pressing while playing.

---

## Features

- Displays all standard keyboard keys.
- Highlights pressed keys in real-time.
- Drag to reposition anywhere on screen.
- Adjustable size, opacity, and colors.
- Multiple animation effects: Lift, Bounce, Rotate, Pulse, Shake, Wobble.
- Gradient support for keys.
- Panel background customization (color, gradient, image).
- Glow and shadow effects.
- Settings saved automatically.
- Keyboard shortcut: Alt+Ctrl+K to toggle overlay.

---

## Installation

1. Install Tampermonkey or Violentmonkey
2. Click [here](https://raw.githubusercontent.com/EverlastingUc/Minefun.io-Visual-Keyboard/main/keyboard.user.js) to install
3. Visit Minefun.io
4. Press Alt+Ctrl+K to toggle the keyboard overlay

---

## How To Use

Press Alt+Ctrl+K to show or hide the keyboard overlay.

### Customization

Open the settings panel by clicking the ⚙️ icon on the keyboard overlay.

| Setting | Description |
|---------|-------------|
| Size | Overall keyboard scale |
| Panel Opacity | Transparency of the keyboard background |
| Key Opacity | Transparency of individual keys |
| Animation | Key press animation style |
| Speed | Animation speed |
| Key Colors | Background and text colors for normal and active keys |
| Gradient | Enable gradient on keys |
| Panel Gradient | Gradient background for the panel |
| Background Image | Custom image for the panel |
| Glow | Neon glow effect on keys |
| Panel Glow | Glow effect on the panel |
| Drop Shadow | Shadow effect on the panel |
